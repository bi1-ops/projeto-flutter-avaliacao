# Projeto Avaliacao

Aplicativo Flutter de gerenciamento de agendamentos para prestadores de servicos, como barbeiros, manicures e outros profissionais autonomos.

## Objetivo

Organizar a rotina de atendimento, permitindo cadastrar servicos e controlar agendamentos de forma simples.

## Fluxo do usuario

1. Criar conta no aplicativo.
2. Fazer login.
3. Adicionar tipos de servicos oferecidos.
4. Adicionar agendamentos para os servicos cadastrados.

## Telas principais

- Tela de cadastro de usuario.
- Tela de login.
- Tela de esqueci senha.
- Tela de CRUD de servicos (criar, listar, editar e excluir).
- Tela de CRUD de agendamentos (criar, listar, editar e excluir).
- Tela de sobre.

## Tecnologias

- Flutter
- Dart
-Supabase
